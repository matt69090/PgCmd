Pour utiliser les programmes fournis :
A) Une seule fois
1)- vous pouvez copier tous les .bat de ce répertoire où vous voulez
2)- vous devez créer un dossier databases où vous voulez (dans vos dossiers personnels) 
    qui contiendra vos bases de données.
	Par defaut un sous-dossier "databases" dans le repertoire contenant ces scripts
3)- vous devez modifier les chemins dans le fichier "defPaths.bat" pour indiquer le répertoire principal des programmes 
    (c:\progsFdB pour les ordinateurs du labo) 
    et votre dossier de base de donnée (celui que vous venez de créer en 2)
4)- vous devez alors exécuter LA PREMIERE FOIS le fichier "initPG.bat" ; un mot de passe vous sera demandé : 
    utilisez un mot de passe spécifique, et pas "important" : 
	il faudra l'indiquer dans vos programmes pour accéder au SGBD ce qui n'est pas "sûr"

B) pour utiliser les programmes :
1) pour démarrer le SGBD (Système de Gestion de Base de Données) postgresl : exécutez "startPG.bat"
2) pour lancer l'interface graphique : exécutez "PgAdmin4.bat"
3) pour stopper le SGBD : exécutez "stopPG.bat"
4) pour lancer netbeans : exécutez "netbeans.bat"
